# Contributing to bananahub-api

## Scope

This repo contains the BananaHub Cloudflare Worker and related docs. Contributions are accepted under this repo's MIT license.

## Commit sign-off

Sign off every commit with the Developer Certificate of Origin:

```bash
git commit -s -m "feat: your change"
```

By signing off, you certify that you have the right to submit the contribution and that it may be distributed under the MIT license.

## Security

- do not commit API keys, GitHub tokens, or real Cloudflare identifiers
- keep telemetry payload examples anonymous and synthetic
